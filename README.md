# Travel Log Map 🇯🇵

地図をタップして旅行先を記録できる、自分用の旅行ログWebアプリです。

## 機能
- 地図をタップ → ピンを追加
- ピンをタップして
  - コメントを保存
  - 写真を最大5枚まで追加（横スクロール）
  - 写真の削除
  - ピンの削除
- データはブラウザに自動保存（localStorage）
- GitHub Pagesで無料公開可能

## 使い方
1. 地図をタップ
2. 「ここにピンを立てる」を押す
3. ピンをタップして写真・コメントを追加

## 技術
- HTML / CSS / JavaScript
- Leaflet.js
- OpenStreetMap
